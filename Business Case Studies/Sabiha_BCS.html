Case Study 1: Insufficient and Inconsistent Water Supply

Background

Residents face daily challenges due to irregular water supply, impacting their basic needs. This issue is exacerbated by outdated infrastructure, poor distribution management, and high demand.

Challenges Identified

  • Outdated Pipelines: Frequent leaks and inefficiencies in distribution.
  • High Demand: Urban growth exceeding supply capacity.
  • Resource Allocation: Poor scheduling of water distribution.

Solution

1. Predictive Maintenance

#include <iostream>
#include <vector>
using namespace std;

void detectLeaks(vector<int> pressureReadings) {
    for (int i = 1; i < pressureReadings.size(); ++i) {
        if (pressureReadings[i] < pressureReadings[i - 1]) {
            cout << "Potential leak detected at segment " << i << endl;
        }
    }
}

int main() {
    vector<int> pressureReadings = {100, 98, 95, 92, 85};
    detectLeaks(pressureReadings);
    return 0;
}

2. Demand Forecasting

#include <iostream>
#include <vector>
using namespace std;

float predictDemand(vector<float> demandHistory) {
    float sum = 0;
    for (float demand : demandHistory) {
        sum += demand;
    }
    return sum / demandHistory.size();
}

int main() {
    vector<float> demandHistory = {150.0, 160.0, 155.0, 158.0};
    cout << "Predicted demand for next day: " << predictDemand(demandHistory) << " units." << endl;
    return 0;
}

Case Study 2: Poor Waste Management

Background

Inefficient waste management practices are leading to pollution and health hazards. Overflowing bins, delayed collection, and lack of segregation exacerbate the problem.

Challenges Identified

  • Delayed Collection: Inefficient scheduling causing overflowing bins.
  • Segregation Issues: Lack of proper sorting at the source.
  • Transportation Inefficiencies: Unoptimized routes for waste trucks.

Solution

1. Route Optimization

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int findShortestPath(vector<vector<int>> &graph, int start, int end) {
    vector<int> distance(graph.size(), INT_MAX);
    distance[start] = 0;

    for (int i = 0; i < graph.size(); ++i) {
        for (int u = 0; u < graph.size(); ++u) {
            for (int v = 0; v < graph[u].size(); ++v) {
                if (distance[u] + graph[u][v] < distance[v]) {
                    distance[v] = distance[u] + graph[u][v];
                }
            }
        }
    }

    return distance[end];
}

int main() {
    vector<vector<int>> graph = {{0, 1, 4}, {1, 0, 2}, {4, 2, 0}};
    cout << "Shortest path: " << findShortestPath(graph, 0, 2) << endl;
    return 0;
}

Case Study 3: Traffic Congestion in Urban Areas

Background

Rapid urbanization has led to increased traffic congestion, resulting in longer commutes, higher pollution levels, and reduced quality of life for residents.

Challenges Identified

  • Peak Hour Traffic: High volume of vehicles during rush hours.
  • Signal Timing: Inefficient traffic light management causing bottlenecks.
  • Emergency Vehicles: Delayed response times due to traffic density.

Solution

1. Dynamic Signal Timing

#include <iostream>
#include <vector>
using namespace std;

void adjustSignalTiming(vector<int> trafficDensity) {
    for (int i = 0; i < trafficDensity.size(); ++i) {
        if (trafficDensity[i] > 50) {
            cout << "Increase green light duration at junction " << i << endl;
        } else {
            cout << "Decrease green light duration at junction " << i << endl;
        }
    }
}

int main() {
    vector<int> trafficDensity = {30, 60, 45, 70};
    adjustSignalTiming(trafficDensity);
    return 0;
}
