Business Case 2: Ensure Access for All to Adequate

SDG Goal Alignment: Ensure Access for All to Adequate is directly aligned with SDG Goal 11.1.

Problem:

  • Goal: Provide efficient routes and infrastructure to ensure access to essential services.

Solution:

<
  • DFS (Depth-First Search): A graph traversal algorithm that explores as far down a branch as possible before backtracking.
  • BFS (Breadth-First Search): A graph traversal algorithm that explores all neighbors at the present depth level before moving on to nodes at the next depth level.

Shortest Path Algorithm

  • Bellman-Ford: An algorithm for finding the shortest paths from a source node to all other nodes in a graph, even when edges have negative weights.

Sorting Algorithms

  • Bubble Sort: A simple comparison-based sorting algorithm that repeatedly steps through the list, compares adjacent items, and swaps them if they are in the wrong order.
  • Selection Sort: A sorting algorithm that repeatedly selects the smallest (or largest) element from the unsorted part of the list and moves it to the sorted part.
  • Merge Sort: A divide-and-conquer algorithm that splits the dataset into smaller subarrays, sorts them, and merges them back into a sorted array.
  • Heap Sort: A comparison-based sorting algorithm that uses a binary heap data structure to build a heap and then sorts the array.
